Лабораторная работа 
Многопользовательские приложения

Реализуйте программу, раздающую индивидуальные задания (с GUI). У преподавателя (сервер) есть список индивидуальных задач (внутри большого TextBox), который он может редактировать, пока не нажмет на кнопку «Готово» («Готово» - означает запустить сервер и начать обслуживать клиентов). Студенты (клиенты) подключаются к преподавателю, получают список оставшихся заданий, выбирают понравившиеся и отправляют преподавателю запрос с ФИО и номером выбранного задания. Если это задание еще никто не выбрал, оно назначается студенту и больше не посылается другим студентам в общем списке заданий. Если же студента кто-то опередил, возвращается ответ – задание уже выбрано. Предусмотрите у преподавателя TextBox для динамического отображения списка номеров оставшихся к данному моменту заданий (можно тот же самый, где преподаватель набирал задания, только нужно из него удалять выбранные студентами задания). У студента должно быть три TextBox: большой для отображения текстов заданий, маленький для ввода номера выбранного задания и TextBox ввода IP-адреса сервера; и кнопка «Отправить запрос».

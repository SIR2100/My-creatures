Лабораторная работа. 
Сетевое программирование на C++.

Реализуйте клиент и сервер для протокола CALC из лекции (TCP и UDP).
Все числа по модулю не превосходят 1 000 000.
Поддерживаемые операции: +, -, *, /, ^
Внимание: в случае UDP пакеты могут теряться! Для тестирования реализуйте искусственные потери некоторых пакетов при пересылке по протоколу UDP.
